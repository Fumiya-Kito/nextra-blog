# 平均との差

- 平均点が整数にならない場合、小数点以下を切り捨てた数値を平均点とします。

## 制約
- 1≦N≦1000
- 0≦Ai≦100
- N,Aiは整数

## 入力
```
N
A1, A2, ..., AN
```

## 出力
- それぞれの人の点数が平均点から何点離れているかを1行ずつ出力してください。
```
1人目の計算結果
2人目の計算結果
...
N人目の計算結果
```

``` cpp copy showLineNumbers filename="解答"
#include <bits/stdc++.h>
using namespace std;

int main() {
  int N;
  cin >> N;

  vector<int> scores(N);
  for (int i = 0; i < N; i++) {
    cin >> scores[i];
  }

  int average = accumulate(scores.begin(), scores.end(), 0) / N;
  for (int i = 0; i < N; i++) {
    cout << abs(scores[i] - average) << endl;
  }
}
```
``` cpp copy showLineNumbers filename="模範解答"
#include <bits/stdc++.h>
using namespace std;
 
int main() {
  int N;
  cin >> N;
 
  // N要素の配列
  vector<int> A(N);
 
  // 入力処理
  for (int i = 0; i < N; i++) {
    cin >> A.at(i);
  }
 
  // 合計点
  int sum = 0;
 
  // 合計点を計算
  for (int i = 0; i < N; i++) {
 
    // ①ここにプログラムを追記
    sum += A.at(i);
 
  }
 
  // 平均点
  int mean = sum / N;
 
  // 平均点から何点離れているかを計算して出力
  for (int i = 0; i < N; i++) {
 
    // ②ここにプログラムを追記
    // 負の数を出力しないように注意
    if (A.at(i) > mean) {
      cout << A.at(i) - mean << endl;
    }
    else {
      cout << mean - A.at(i) << endl;
    }
 
  }
 
}
```
