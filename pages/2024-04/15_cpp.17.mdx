# 計算量
## キーポイント
- プログラムを実行するときには処理内容に応じた実行時間がかかる
- コンピュータの記憶領域(メモリ)は有限であり、プログラムで変数を使用した分だけメモリを消費する
- プログラムの実行時間・メモリ使用量が入力に応じてどのように変化するかを見積もったものを、それぞれ時間計算量・空間計算量という
- 計算量の表記にはオーダー記法を用いることが多い


## オーダー記法 O(・)
時間計算量や空間計算量の表現として、オーダー記法が用いられることが多い。
厳密に計算量を計算するのは難しいためNを大きくしていったときに一番早く発散する項目`X`のみを抽出して`O(X)`と書く
- `3N^2+7N+4`という式はオーダー記法では`O( N^2 )`
- `3 + 4` → `O(1)`
- `N^2 + 2^N` という式は`O(2^N)`
- 以下のプログラムは`O(N^2)`
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  int N;
  cin >> N;

  int sum = 0;

  for (int i = 0; i < N; i++) {
    for (int j = 0; j < N; j++) {
      sum += i * j;
    }
  }

  for (int i = 0; i < N; i++) {
    sum += i;
  }

  for (int i = 0; i < N; i++) {
    sum *= i;
  }

  cout << sum << endl;
}

```
- 次のプログラムはNが2で何回割れるかを数えるものです。 このプログラムの計算量は`O(logN)`です。
- ループする回数が大体`log2N`回になる
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  int N;
  cin >> N;
  int cnt = 0;
  while (N > 0) {
    cnt++;
    N /= 2;
  }
  cout << cnt << endl;
}

```

```
O(1)<O(logN)<O(N)<O(NlogN)<O(N^2)<O(2^N)
```
