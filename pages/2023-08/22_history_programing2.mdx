# プログラミング言語の歴史2

### オブジェクト指向
- simulaとLISPから影響を受けたsmalltalkという言語が初めてオブジェクト指向を取り入れた
- 1983年になると、オブジェクト指向の便利さが認められ、(Cにオブジェクト指向取り入れた)C++が誕生
- Objective-C (Smalltalk言語の中にC言語をまるっと合体)

### パソコン
- 1976年: スティーブ・ジョブズとウォズニアックがApple Computerを設立
- 一般家庭向けコンピュータの普及を目指す → Apple2が大ヒット
- IBMはこれを受けPC作成に取り組むが、OS開発に苦しみ、をMSに開発を依頼

### GUI
- 1985: CUIからGUIを取り入れる動きが始まる
- macintosh, Windows OS が誕生

### Web
- 1986年: CSNET(アメリカ), JUNET(日本)が繋がり、国を超えたネットワークができ始める
- 開発スピード競争が激しくなる背景もあり、スクリプト言語(インタープリタ型)であるPerlなどが台頭
- 1991年: Simple is Bestを重点に置き、「開発者によってコードのばらつき」を抑えたPythonが誕生
- 同時期にMSはWindowsアプリ向けにVisualBasicを発表（アイデアは役割を特化させることで開発者のばらつきを抑えた）
- Unixと同じように動くOS Linuxが誕生、OSSとして無償配布
- CERNのティム・バーナーズ・リーはインターネット上でつながるページを作成
  - WWWと名付け、HTML言語のハイパーリンク、だれでも使えるように


### ブラウザ
- 1994年: インタネットで商用利用が認められる（小さい単位の〇〇NETはもはや消滅）
- インターネットの窓口であるブラウザの開発競争が始まる
- Netscape Navigator(ネスケ)が先行する
- MSがInternet Exprolerを発表し第一次ブラウザ戦争


### JavaとJVM
- C++を参考にJavaが開発
- コンピュータやメーカーや機種に合わせて開発する必要のないようにJVMを発表（大成功, トップ言語に）
- Netscapeと提携し、Javaアプレットを発表
- Netscape社はJavaとは別に、ブラウザ専用のスクリプト言語LiveScriptを開発
- Java人気にあやかって、JavaScriptと名称変更

### サーバー用
- Webブラウザとインターネットの普及によって開発競争が進んでいるが、サーバー側の言語も必要
- 1995年: Perlが多かったが、Webに特化したPHPが開発
- Apache, MySQLがOSSとして発表
- LAMP(Linux, Apache, MySQL, PHP(Python, Perl))の無料組み合わせが完成
- Rubyが日本で誕生（徹底的なオブジェクト指向、とにかく楽しく書けるがモットー）

### 第一次ブラウザ戦争
- 1997年: Netscapeが言語を統一しようとして、Ecma Internationalに仲裁を依頼しECMAScript（ブラウザで使う言語のルール）が定められる
- 1998年: MSがWindowsにIEを標準搭載することで第一次ブラウザ戦争に終止符が打たれる
- MS由来のJSは評判がよくなく、動くサイトをつくるならFlash(後にAdobeに買収)という技術が流行

### 2000年問題
- FORTRANやCOBOLでは、西暦を下二桁で表す手法が使われていたため、2000年=1900年とコンピュータ勘違いする問題が指摘
- 2000年のうるう年が特殊なこともあり、心配されていたが、そこまで大変なことにはならなかった
- 2000年になるとグラフィック性能が上がりGPUが搭載されるように。専用の言語も開発
- C, C++, Javaといった覇権言語をMSは自社のものしたいと考えC#を開発(C++++というのが由来)

- すべての元

- 家電、その他全般
  - Java
- Windows
  - VB, VBA
