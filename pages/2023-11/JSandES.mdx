# JavaScript と EcmaScript

Jsの歴史

1990年代インタネット黎明期
Netscape navigator.  Vs.  internet explorer 

でブラウザ戦争をしていた際に、JavaScriptを作成

この2つに互換性がなかったから、ECMAscriptを作り複数のブラウザ上で実行できるように**仕様を決定しようとした**

結果、ブラウザ間での仕様を統一に成功。言語ではなく、EcmaScriptという仕様。
JavaScript のコアな部分を仕様がEcmaScriptとなる

Java Scriptの中には、ECMAScriptが必ず含まれていて、web環境では、webAPIをnode 環境では、…と環境によって使える機能が変化する
