# TypeScript4

## Core Types
|Type|Example|Description|
|---|---|---|
|number|1, 5.3, -2|すべての数値, int, float, double等区別しない|
|string|'Hi', "hi", \`HI\`|すべてのテキスト値|
|boolean|true, false|２つのみ、"truthy", "falsy"の意味ではない|
|object|\{age: 30\}|どんなJS Objectでも良いが、オブジェクトの"型"を指定できる|
|Array|[1, 2, 3]|どんなJS Arrayでも良いが、要素の型は指定しても良いし、柔軟に型を取ることも可能|
|Tupple|[1, 2]|JSにはなくTSで追加された型、”長さ固定”配列|
|Tupple|[1, 2]|JSにはなくTSで追加された型、”長さ固定”配列|

## number, string, boolean
```ts filename="app.ts"
function add(n1: number, n2: number, showResult: boolean, phrase: string) {
  if (showResult) {
    console.log(phrase + `${n1+n2}`);
  } else {
    return n1 + n2;
  }
}

const number1 = 5;
const number2 = 2.8;
const printResult = true;
const resultPhrase = 'Result is: ';

add(number1, number2, printResult, resultPhrase);
```
