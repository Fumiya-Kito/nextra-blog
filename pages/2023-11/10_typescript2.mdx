# TypeScript 

### TSをインストール
TSというプログラミング言語とともにコンパイラも一緒にインストールされる
- npmを使う(globalにインストールする)
<br/>→`npm install -g typescript`
- そのほか
<br/>→ [TypeScript.org](https://www.typescriptlang.org/download)

### TSをコンパイル
```ts filename="example.ts"
const button = document.querySelector("button")!;
const input1 = document.getElementById("num1")! as HTMLInputElement;
const input2 = document.getElementById("num2")! as HTMLInputElement;

function add(num1: number, num2: number) {
  return num1 + num2;
}

button.addEventListener("click", function() {
  console.log(add(+input1.value, +input2.value));
});
```
- `!`をつけることで、`null`の可能性を否定
- `as`をでHTML要素の型を指定できる

↓ コンパイル`tsc example.ts`
```js filename="example.js"
var button = document.querySelector("button");
var input1 = document.getElementById("num1");
var input2 = document.getElementById("num2");
function add(num1, num2) {
    return num1 + num2;
}
button.addEventListener("click", function () {
    console.log(add(+input1.value, +input2.value));
});
```

### 重要なのはHTMLには`.js`ファイルを読み込むこと
```html {8} filename="index.html"
<!DOCTYPE html>
<html lang="en">
  <head>
```
