# v0を使ってレイアウトを作成
NextJSでダークモードとライトモードを切り替える方法←今回はやめた

### `ThemeProvider`とは 
```
`ThemeProvider`は、Next.jsアプリケーションでダークモードやライトモードなどのテーマ切り替え機能を提供するコンポーネントです。shadcn/uiライブラリの一部として提供されており、アプリ全体のテーマを管理するために使用されます。
```
### プロパティ
1. `attribute="class"` - テーマの切り替え方法を指定します。`"class"`を指定すると、HTMLのクラス属性を使用してテーマを切り替えます。
2. `defaultTheme="light"` - デフォルトのテーマを指定します。`"light"`、`"dark"`、または`"system"`（システム設定に従う）を指定できます。
3. `enableSystem={true}` - システムのテーマ設定（OSの設定）に従うかどうかを指定します。
4. `disableTransitionOnChange={true}` - テーマ切り替え時のトランジション効果を無効にするかどうかを指定します。

```txt filename="next-themesとは"
next-themes は、Next.js フレームワークにおけるテーマ管理を簡単にするためのライブラリです。
next-themes を使用することで、ダークモードやライトモードなど、異なるテーマを簡単に切り替えることができます。
```


- `components/theme-provider.tsx`を作成(手動)
```tsx filename="components/theme-provider.tsx"
"use client"

import * as React from "react"
import { ThemeProvider as NextThemesProvider } from "next-themes"
import { type ThemeProviderProps } from "next-themes"

export function ThemeProvider({ children, ...props }: ThemeProviderProps) {
  return <NextThemesProvider {...props}>{children}</NextThemesProvider>
}
```
あとトグルなどを設定すれば実装完了↓
